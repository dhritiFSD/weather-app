# 🌦️ Weather App

A simple and responsive Weather Application built using HTML5, CSS3, JavaScript, and TypeScript.
This app allows users to search for any city and get real-time weather information using the OpenWeather API.

---

## 🚀 Features

* Search weather by city name
* Real-time temperature display
* Weather condition display
* Humidity information
* Responsive and clean UI
* API integration using Fetch API
* Built with TypeScript

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* TypeScript
* OpenWeather API

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/dhritiFSD/weather-app.git
```

---

### 2️⃣ Open Project Folder

```bash
cd weather-app
```

---

### 3️⃣ Install TypeScript

```bash
npm install -g typescript
```

---

### 4️⃣ Create OpenWeather API Key

Create free account from:

https://openweathermap.org/api

Copy your API key and replace:

```typescript
const apiKey: string = "YOUR_API_KEY";
```

with your actual API key in `app.ts`.

---

### 5️⃣ Compile TypeScript

```bash
tsc
```

This will generate:

```bash
dist/app.js
```

---

### 6️⃣ Run the Project

Open `index.html` in browser

OR use Live Server extension in VS Code.
